# LiveCD

My custom USB with LiveCD / installers layout:

1. 20G Encrypted partition - for sensitive data (gnupg keys, recovery keys) backup.
2. 1G EFI partition - for loading Windows installer.
3. 9G NTFS - Windows11 installer.
4. 20G EFI partition - for Arch Linux (and other future linuxes/tools).
5. (optionally) 60G - data files volume (ntfs/exfat)
